Estructura de una página web

¿Cómo es?

En JavaScript, la estructura se refiere a cómo se organizan y se dividen las diferentes partes del código fuente en un archivo JavaScript. A continuación, se muestra la estructura básica de un archivo JavaScript:

Declaraciones de variables: es común comenzar un archivo JavaScript con la declaración de variables que se utilizarán en el código. Las variables pueden declararse usando las palabras clave "var", "let" o "const".

Funciones: las funciones en JavaScript se definen usando la palabra clave "function". Las funciones pueden tener parámetros y devolver valores.

Eventos: JavaScript se utiliza a menudo para agregar interactividad a una página web. Los eventos son acciones que ocurren en la página, como hacer clic en un botón o mover el mouse, y se manejan utilizando funciones de JavaScript.

Métodos y propiedades: los objetos en JavaScript tienen métodos y propiedades. Los métodos son funciones que se aplican a un objeto, mientras que las propiedades son valores que se almacenan en un objeto.

Condicionales y bucles: JavaScript proporciona estructuras de control de flujo, como condicionales ("if", "else") y bucles ("for", "while"), para ejecutar ciertas partes del código en función de ciertas condiciones.

Manipulación del DOM: El DOM (Document Object Model) es la estructura de objetos que representa una página web en el navegador. JavaScript se utiliza a menudo para manipular el DOM, agregando o eliminando elementos y cambiando el contenido de la página.


Funciones de JavaScript

¿Cuales son?

JavaScript se puede ejecutar en HTML de varias formas, la forma más común son los eventos del DOM (Document Object Model)

Algunos ejemplos:

onload: Se ejecuta cuando la página web completa, incluidos sus recursos, se ha cargado en el navegador.

onclick: Se ejecuta cuando un elemento se hace click.

onmouseover: Se ejecuta cuando el puntero del ratón se mueve sobre un elemento.

onsubmit: Se ejecuta cuando se envía un formulario.

onkeydown: Se ejecuta cuando se presiona una tecla del teclado.

onresize: Se ejecuta cuando la ventana del navegador se redimensiona.
